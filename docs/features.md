# Features

This document provides a comprehensive list of all features in WebsiteBuilderAI, organized by status and category.

## Feature Status Legend

| Status | Description |
|--------|-------------|
| Active | Fully implemented and available |
| Beta | Working but may have rough edges |
| Coming Soon | Planned for near-term implementation |
| Planned | On the roadmap but not yet scheduled |

## Content Management

### Active Features

#### Add Section
Add new sections to your website including:
- Hero (main banner)
- About (company/personal info)
- Features (product/service features)
- Services (service offerings)
- Testimonials (customer reviews)
- Team (team member profiles)
- Pricing (pricing tables)
- Contact (contact form/info)
- CTA (call-to-action)
- Gallery (image gallery)
- FAQ (frequently asked questions)
- Blog (blog posts preview)
- Newsletter (email signup)
- Stats (statistics/metrics)
- Portfolio (work showcase)
- Menu (restaurant menu)

**Trigger words**: "add", "new section", "insert", "create section"

#### Edit Section
Modify existing section content:
- Change titles and subtitles
- Update paragraph text
- Modify list items
- Edit testimonials, features, pricing, etc.

**Trigger words**: "edit", "change", "update", "modify", "fix", "rewrite"

#### Remove Section
Delete sections from the website.

**Trigger words**: "remove", "delete", "hide", "get rid of"

#### Reorder Sections
Change the order of sections on the page.

**Trigger words**: "move", "reorder", "rearrange", "swap"

#### Get Site Information
Retrieve current website state, sections, and configuration.

**Trigger words**: "what sections", "show me", "current state"

## Media & Assets

### Active Features

#### Generate Image with AI
Create custom images using DALL-E 3:
- Hero images
- Section images
- Custom prompts
- Multiple styles (photo, illustration, abstract)

**Requirements**: OpenAI API key, Pro plan

**Trigger words**: "generate image", "create image", "ai image"

#### Generate Logo with AI
Create logo variations using DALL-E 3:
- Minimalist, Bold, Vintage, Playful, Corporate styles
- Multiple variations option
- Custom color matching

**Requirements**: OpenAI API key

**Trigger words**: "logo", "brand mark", "create logo"

#### Upload Assets
Upload your own images and files:
- Images (JPG, PNG, GIF, WebP)
- Logos
- Favicons
- Videos
- Documents

**Requirements**: S3/R2 storage configuration

**Trigger words**: "upload", "my image", "add image"

#### Suggest Images
Get AI suggestions for images that would work well in a section.

**Trigger words**: "suggest images", "image ideas"

### Beta Features

#### Generate Video with AI
Create short videos using Runway ML Gen-3:
- Hero background videos
- Image animation
- Custom prompts
- Multiple durations (5s, 10s)
- Various aspect ratios

**Requirements**: Runway API key, Enterprise plan

**Trigger words**: "video", "animation", "motion"
